Tech Stack:
===========
Frontend:  React 19, Vite, Tailwind CSS, React Router, Lucide Icons
Backend:   FastAPI, SQLAlchemy, PostgreSQL, Redis, Celery, Web3.py
Database:  PostgreSQL with async support
Cache:     Redis
Blockchain: Web3.py for smart contract interaction
Container: Docker & Docker Compose

Quick Start Commands:
====================
Backend:
  cd backend
  python -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  uvicorn app.main:app --reload

Frontend:
  cd frontend
  npm install
  npm run dev

Docker:
  docker-compose up

Project URLs (when running locally):
====================================
Frontend:     http://localhost:5173
Backend API:  http://localhost:8000
API Docs:     http://localhost:8000/docs
